# playerFuncs.emit.createShard

Create a simple full-screen text shard.

_Accessible on Server Side_

# Example

```typescript
playerFuncs.emit.createShard(player, {
    duration: 5000,
    title: '~r~Hello World!',
    text: '~y~Shards are pretty neat sometimes.',
});
```
