# playerFuncs.emit.createCredits

Create a simple credit overlay on screen.

_Accessible on Server Side_

# Example

```typescript
playerFuncs.emit.createCredits(player, {
    duration: 5000,
    role: 'Athena Creator',
    name: 'Stuyk',
});
```
